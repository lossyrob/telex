# Design and internals

This guide covers using and operating telex. The design and implementation are
documented separately in the repository:

- [DESIGN.md](https://github.com/lossyrob/telex/blob/main/docs/design/DESIGN.md):
  the working design.
- [DECISIONS.md](https://github.com/lossyrob/telex/blob/main/docs/design/DECISIONS.md):
  the decision log (ADRs).
- [ARCHITECTURE.md](https://github.com/lossyrob/telex/blob/main/docs/design/ARCHITECTURE.md):
  the component architecture.
- [daemon.md](https://github.com/lossyrob/telex/blob/main/docs/design/daemon.md):
  the local exchange daemon.
- [copilot-bridge-push.md](https://github.com/lossyrob/telex/blob/main/docs/design/copilot-bridge-push.md):
  Copilot CLI push delivery.

The forward-looking discovery and dispatch proposals live in
[DISPATCH.md](https://github.com/lossyrob/telex/blob/main/docs/design/proposals/DISPATCH.md) and
[EXTENSIONS.md](https://github.com/lossyrob/telex/blob/main/docs/design/proposals/EXTENSIONS.md).
