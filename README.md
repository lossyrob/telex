# Telex

Telex is a message fabric for AI agent sessions: a small, durable, addressable way
for agents to send coordination messages, field reports, questions, handoffs, and
disposition-required work across local and networked execution environments.

The name is intentional. Historical telex was not casual chat. It was a serious
business text network for machines: addresses, receipts, terse messages, and
durable records. That makes it a fitting metaphor for the early AI era, where
capable but ephemeral agents need protocol, identity, delivery semantics, and
audit trails before large-scale autonomous coordination can become reliable.
